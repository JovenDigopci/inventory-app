# Requirements Index

This folder contains the requirements documentation for the decant perfume inventory management system.

Files:

- `00-research-summary.md` - researched inventory practices, decant-specific rules, formulas, and sources.
- `01-business-requirements-document.md` - business problem, goals, scope, stakeholders, and success metrics.
- `02-user-stories.md` - user stories and acceptance criteria.
- `03-functional-requirements.md` - functional system behavior required for development.
- `04-non-functional-and-system-requirements.md` - technology, security, performance, backup, and system requirements.
- `05-data-model.md` - proposed database entities and data rules.
- `06-implementation-plan.md` - phased development plan and release checklist.

Approved stack for MVP:

- Frontend: HTML, CSS, JavaScript.
- Backend: Node.js with Express.
- Database: MySQL Community Server.
